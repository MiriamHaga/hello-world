Lista numerada

Digite os itens da lista. Dê um ENTER para cada novo item.

Código html

Copie o código abaixo e cole no lugar correspondente no curso.

<ul class="list-group list-group-numbered">
<li class="list-group-item">
<span class="list-group-item__number">{{ $index+1 }}</span>{{ listNumberedItem }}
</li>
</ul>

Construtor de elementos (gerador de html)

  • {{ $index+1 }}{{ listNumberedItem }}
