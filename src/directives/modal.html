Modal

Caso tenha mais de um elemento como este na mesma tela é necessário definir um novo id.

Código html

Copie o código abaixo e cole no lugar correspondente no curso.

<a type="button" class="sr-only-focusable" data-toggle="modal" href="#Janela{{ modalId }}" role="link">{{ modalLink }}</a>
<div class="modal fade" id="Janela{{ modalId }}" tabindex="-1" role="dialog" aria-labelledby="ConteudoJanela{{ modalId }}">
<div class="modal-dialog modal-smmodal-lg" role="document">
<div class="modal-content">
<div class="modal-header">
<button type="button" class="close" data-dismiss="modal" aria-label="Fechar"><span aria-hidden="true">×</span></button>
<h4 class="modal-title" id="ConteudoJanela{{ modalId }}">{{ modalTitle }}</h4>
</div>
<div class="modal-body">
<p>{{ modalText }}</p>

<!--Manter o botão no fim do conteúdo para acessibilidade-->
<div class="clearfix"></div>
<button type="button" class="sr-only sr-only-focusable btn btn-default pull-right" data-dismiss="modal">Fechar</button>
<div class="clearfix"></div>
</div>
</div>
</div>
</div>
