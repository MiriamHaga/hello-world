Introdução

Esse construtor foi projetado com o objetivo de facilitar a replicação do código html dos elementos, especialmente os interativos.

O código base de estilo e os scripts já devem ter sido criados anteriormente como modelo, esse construtor auxilia apenas na identificação e montagem. O css e os scripts do curso se encontram em uma pasta específica no projeto e podem ser substituídos para visualização. Porém, essa substituição não é necessária, pois a maioria dos cursos utiliza uma estrutura padrão para esses elementos, então o html gerado deve incorporar os estilos adequadamente quando "colado" no projeto. Além disso, se o css do elemento no curso vincular alguma imagem de background, por exemplo, ela não aparecerá no preview.

Caso a estrutura html projetada para o elemento no curso não siga o padrão desse construtor ele não irá funcionar, sendo necessário ajustar a ferramenta ou o css do curso.

Utilização

  1. Selecione no menu o elemento que precisa criar.

  2. Preencha os campos de acordo com as especificações do elemento que deseja criar.

  3. O campo html irá se atualizar de acordo com os itens preenchidos. Copie o código e cole no local onde o elemento deve ser encaixado no projeto.
