Introdução

Esse construtor foi projetado com o objetivo de facilitar a replicação do código html dos elementos, especialmente os interativos.

O código base de estilo e os scripts já devem ter sido criados anteriormente como modelo. Esse construtor auxilia apenas na identificação dos elementos e montagem. O css, imagens e scripts do curso se encontram em uma pasta específica no projeto e podem ser substituídos para uma prévia de visualização (em razão de estar incorporada no construtor, o preview pode não ser totalmente fiel). Porém, essa substituição não é necessária, pois a maioria dos cursos utiliza uma estrutura padrão para esses elementos, então o html gerado deve incorporar os estilos adequadamente quando "colado" no projeto.

Nessa versão do construtor, os elementos foram baseados na estrutura dos últimos cursos do Poupatempo: Reconhecimento de paternidade, Atestado de Antecedentes Criminais e Inscrição de matrícula.

Caso a estrutura html projetada para algum dos elementos do curso não siga o padrão desse construtor, ele não irá funcionar adequadamente, sendo necessário ajustar a ferramenta ou o css do curso.

Utilização

  1. Selecione no menu lateral o elemento que precisa criar.

  2. Preencha os campos de acordo com as especificações do elemento que deseja criar.

  3. O campo html irá se atualizar de acordo com os itens preenchidos. Copie o código e cole no local onde o elemento deve ser encaixado no projeto.
